Project Description
The Hospital Management System is a web-based solution designed to streamline healthcare administration. Building on my coursework in Web Design and Development (CSI-506) and Database Systems (CSI-408), this project automates patient records, appointment scheduling, and staff management to improve operational efficiency.


Key Features
Patient Management: Registration and digital record-keeping.

Doctor Scheduling: Managing appointments and availability.

Administrative Dashboard: Overview of hospital statistics and data management.

Secure Authentication: Role-based access for admins, doctors, and staff.

Technical Stack
Frontend: HTML5, CSS3, JavaScript.

Backend: PHP / MySQL (as per database requirements).
